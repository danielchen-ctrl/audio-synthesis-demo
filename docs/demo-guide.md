# 语料生成平台 — 使用说明

> 版本：2026-05-03  
> 服务地址：`http://127.0.0.1:8899/`

---

## 快速启动

```bash
# 方式一：命令行
python server.py

# 方式二：双击批处理
start_platform.bat
```

启动后浏览器访问 `http://127.0.0.1:8899/`。

---

## 界面导航

顶部导航栏包含五个页面：

| 按钮 | 功能 |
|------|------|
| 🏠 全部文件 | 查看所有已生成/上传的音频文件 |
| 📁 我的文件 | 按文件夹分类管理文件 |
| ⚙️ 生成任务 | 查看后台任务队列和进度 |

右上角 ☀️/🌙 按钮切换亮色/暗色主题。

## 生成对话语料

### 1. 打开生成弹窗

点击顶部「✨ 生成语料」按钮，或移动端汉堡菜单中同名按钮。

### 2. 选择生成模式

| 模式 | 适用场景 |
|------|---------|
| **AI 生成文本** | 由 LLM 自动生成对话内容，再合成音频 |
| **手动输入文本** | 自行粘贴对话文本，仅做音频合成 |

---

### 模式一：AI 生成文本

#### 步骤

1. **选择模板**：从下拉框选择 18 个行业场景之一（如"医疗健康｜慢病随访"），或选"自定义模板"手动填写。

2. **输入主题**（两种方式）：
   - **自由输入**：在文本框中直接描述对话背景
   - **预置主题**：点击「📋 预置主题」按钮，从 22 个内置场景中选择，自动填充角色和关键词

3. **配置参数**：

   | 参数 | 说明 |
   |------|------|
   | 语言 | 中文 / English / 日本語 / 한국어 / 粤语 等 |
   | 字数 | 300 / 500 / 1000 / 1500 / 2000 / 3000 / 5000 字 |
   | 关键词 | 逗号分隔，LLM 会在对话中自然嵌入这些词 |
   | 发言人数 | 2–6 人 |

4. **点击「✨ 根据以上配置生成文本」**：等待 LLM 生成对话文本（约 15–60 秒）。

5. **预览并调整**：生成完成后在预览区查看文本，可点击「🔄 重新生成」重试。

6. **点击「🎙 合成音频」**：为每个发言人选择音色，点击确认合成。

7. **下载/管理**：合成完成后文件自动出现在「全部文件」页面。

---

### 模式二：手动输入文本

1. 选择「手动输入文本」模式卡片。
2. 选择语言和发言人数。
3. 在文本框中按如下格式粘贴对话：
   ```
   Speaker 1: 您好，请问您最近血压控制情况如何？
   Speaker 2: 最近一直在吃药，但是血压还是偶尔偏高。
   Speaker 1: 您的用药依从性怎么样？
   ```
4. 点击「🎙 合成音频」选择音色并合成。

## 文件管理

### 全部文件页

- **搜索**：按文件名、语言、场景筛选
- **排序**：按时间（最新/最旧）、文件名升降序
- **操作**：播放预览 / 下载 / 查看详情 / 移入回收站
- **批量操作**：勾选多个文件后，可批量移动到文件夹或删除

### 我的文件页

- 点击「+ 新建文件夹」创建分类文件夹
- 点击文件夹名进入，查看该文件夹内的文件
- 上传音频时可在弹窗中选择目标文件夹

### 回收站

- 已删除文件在此保留 30 天内可还原
- 点击「↩ 还原」恢复单个文件
- 点击「🗑 永久删除」彻底清除，不可恢复
- 支持批量还原和批量永久删除

## 上传音频

1. 点击「⬆ 上传音频」按钮。
2. 选择文件（支持 MP3、WAV、M4A、FLAC 等格式）。
3. 填写元数据：语言、发言人数、场景、主题、目标文件夹（可选）。
4. 点击确认上传，文件出现在「全部文件」和所选文件夹中。

---

## 生成任务队列

「⚙️ 生成任务」页面展示所有通过任务系统提交的生成请求：

| 状态 | 含义 |
|------|------|
| queued | 等待执行 |
| generating_text | LLM 正在生成文本 |
| synthesizing | 正在合成音频 |
| completed | 完成，可在全部文件中查看结果 |
| failed | 失败，悬停错误图标可查看原因 |

任务进行中时页面每 3 秒自动刷新。点击「重试」可对失败任务重新执行。

## 常见问题

**Q: 生成文本卡住不动？**  
LLM 推理在 bundle 内运行，较慢是正常现象（1500 字约需 30–90 秒）。若超过 3 分钟请重试。

**Q: 合成音频报错？**  
检查 `bin/ffmpeg.exe` 是否存在，以及网络是否能访问 edge-tts 服务（需要境外网络或代理）。

**Q: 文件上传后在文件夹中看不到？**  
确认上传时已在弹窗中选择目标文件夹；若未选择则文件归入"无文件夹"，在全部文件页可见。

**Q: 重新生成没有反应？**  
点击确认弹窗的「确认」按钮即可。确认后重新生成会替换当前预览文本。

**Q: 如何修改预置主题内容？**  
编辑 `demo/预置对话情景参数.txt`，每个条目以 `N） 行业｜场景名` 开头，重启服务器生效。

## 技术依赖

| 组件 | 用途 |
|------|------|
| Python 3.11+ | 运行环境 |
| Tornado | Web 服务框架 |
| edge-tts | 微软 TTS 语音合成 |
| pydub | 音频时长探测 |
| ffmpeg | 音频片段拼接（`bin/ffmpeg.exe`） |
| SQLite | 平台数据库（`runtime/platform.db`） |
| SceneDialogueDemo.exe | PyInstaller 打包的 LLM 推理引擎 |
